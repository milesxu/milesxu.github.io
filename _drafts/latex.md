# LaTex 数学公式

## 标记

```
+ - = ! / ( ) [ ] < > | ' :
```

除了以上列出的，还有许多标记需要通过发出不同的命令来完成。诸如希腊字符、集合和关系标记、箭头、二元操作符，等等。   
例如：   

```
\forall x \in X, \quad \exists y \leq \epsilon
```
$$ \forall x \in X, \quad \exists y \leq \epsilon $$
